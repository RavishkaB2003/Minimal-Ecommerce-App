
# Nike E-Commerce Mobile App

## A Flutter UI & State Management Project

This project is a minimal inspired e-commerce mobile application developed using Flutter. It focuses on clean UI design, reusable components, and efficient state management while simulating a real-world shopping experience with a shop and cart flow.

The app was built as a learning project to strengthen Flutter fundamentals, widget composition, and Provider-based state management.
